# Very-Very-Bad-Calculator
A Calculation Programm, its my first Programm that will have a GUI so it could end up looking a bit weird.




What it can do:
-Perform Calculations using the 4 Basic Operators (+)(-)(*)(/)




What it cant do:
-Perform more complex calculations (I will try to implement more in Future Versions)




How to Run it:
  1. Download the Zip-File from the lastest Release
  2. Unpack the Zip-File
  3. Launch the Exe-File in the Folder




How to Uninstall it:    
-Delete the Folder with the Exe-File in it
